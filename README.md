# gx_app_3

# 概要
情報を逃さずインプットの定着を図るアプリです。 
インターネットにある記事を保存できるあとで読む系のサービスに近いですが、記事の内容を覚えるためにメモを追加できたり、たまった記事を消化できるようにリマインド設定できたりします。
